* ### Regex Metacharacters 

* .	 : Any character (may or may not match terminator)
* \d : Any digit, [0-9]
* \D : [^0-9] or [^\d]

* \s : Any whitespace character, [\t\n\x0B\f\r]
* \S : [^\s]

* \w : Any word character, [a-zA-Z_0-9]
* \W : [^\w]

* \b : A word boundary
* \B : [^\b]

* ([a-z]|[1-9]){1} : Atleast 1 char OR 1 integer
* ? : Optional
